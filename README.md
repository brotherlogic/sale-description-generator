# sale-description-generator

This project is an API to take in (a) user notes about record condition and (b) the records actual condition levels (media / sleeve) and then generate a sale description that captures this information in a positive listing.

## Sources

The Discogs condition manual:

https://www.discogs.com/selling/resources/how-to-grade-items


Goldmine standard:

https://www.goldminemag.com/collector-resources/record-grading-101/

These form the basis of the generated description, but the project pulls on this and the notes provided to build out the overall description.

The description itself should be around a long paragraph.
